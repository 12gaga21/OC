## 🌟 О Проекте

**«Священный Ритуалъ»** — это уникальная операционная система, полностью написанная на **ассемблере NASM**, сочетающая в себе:
- ✝️ **Эстетику Российской Империи** (дореформенная орфография, золотые церковные рамки)
- ⚙️ **Техно-ритуал Адептус Механикус** (зелёный текст, бинарные молитвы)
- 🇷🇺 **Дух Русского Православия** (символы †, обращения к Царю-Машине)

### 🎯 Ключевые Возможности

| Возможность | Описание | Статус |
|-------------|----------|--------|
| **Монолитное Ядро** | Чистый ассемблер, без зависимостей | ✅ Готово |
| **Многозадачность** | 16 задач, вытесняющий планировщик | ✅ Готово |
| **Файловая Система** | FAT16/32 с журналированием и кэшированием | ✅ Готово |
| **Сеть TCP/IP** | Ethernet, IP, TCP, UDP, ICMP, HTTP, DNS | ✅ Готово |
| **Безопасность** | Модуль «Святая Инквизиция» (проверка подписей) | 🔄 В разработке |
| **Библиотека SYSLIB** | Единая библиотека функций для всех программ | ✅ Готово |
| **Русский Язык** | Кодировки CP866/CP1251, раскладка ЙЦУКЕН | ✅ Готово |
| **TUI Интерфейс** | Оконный интерфейс в золотых рамках | ✅ Готово |
| **Загрузка Программ** | Формат .BIN, загрузчик с проверкой целостности | 🔄 Интеграция |
| **HAL (Абстракция)** | Работа на любом x86 устройстве (ЧПУ, контроллеры) | 🔄 В разработке |

## 🖥️ Визуальный Стиль «Злато-Изумрудный Ритуалъ»

```
† † † ВСЕМОСТИВЫЙ ГОСУДАРЬ † † †
╔════════════════════════════════════════╗
║  [СЛАВЯНСКІЙ ТЕРМИНАЛЪ]> ls            ║
║                                        ║
║  Содержимое каталога /ROOT:            ║
║  † KERNEL.BIN   (Священное Ядро)       ║
║  † SYSLIB.SYS   (Златой Кодексъ)       ║
║  † HELLO.BIN    (Первый Духъ)          ║
║                                        ║
║  Слава Омниссіи!                       ║
╚════════════════════════════════════════╝
```

- 🟢 **Изумрудный текст** (0x0A) — эстетика Механикуса
- 🟡 **Золотые рамки** (0x1E) — стиль Российской Церкви
- 🔴 **Красные ошибки** (0x0C) — предупреждения Инквизиции
- 🔵 **Темно-синий фон** (0x01) — классический терминал
- ✝️ **Символы крестов** (†) — вместо обычных углов
- 📜 **Дореформенная орфография** (ъ, ѣ, і, ѳ)

## 🚀 Универсальность

ОС работает на **любом x86-совместимом устройстве**:
- 💻 Персональные компьютеры (386+)
- 🏭 Промышленные ЧПУ станки
- 🎮 Встраиваемые контроллеры
- 📡 Сетевые устройства
- 🤖 Робототехника

**Минимальные требования:**
- Процессор: Intel 80386 или совместимый
- Память: 512 KB RAM
- Накопитель: 1 MB (FDD/HDD/SSD/CF)
- Видео: VGA совместимый адаптер

## 🛠️ Сборка и Запуск

### Требования
- NASM 2.15+
- QEMU 6.0+ (для тестирования)
- Linux/macOS

### Команды
```bash
cd os/build
./build.sh           # Сборка образа os.img
qemu-system-i386 -fda build/os.img  # Запуск в эмуляторе
```
