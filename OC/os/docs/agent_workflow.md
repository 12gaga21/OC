# Детальный план работы ИИ-агента по созданию ОС на ассемблере

## 1. Этапы создания операционной системы на ассемблере

### 1.1. Инициализация проекта
- Определение целевой архитектуры (x86, ARM, RISC-V)
- Создание структуры проекта с каталогами для исходного кода, сборки и тестов
- Настройка инструментов разработки (ассемблер, линковщик, эмулятор)

### 1.2. Разработка загрузчика (Bootloader)
- Создание загрузочного сектора (512 байт)
- Реализация базовой инициализации процессора
- Настройка работы с диском и памятью
- Переход к ядру ОС

### 1.3. Создание ядра ОС
- Инициализация защищенного режима (для x86)
- Настройка управления памятью
- Реализация диспетчера задач
- Создание системы прерываний
- Реализация базовых системных вызовов

### 1.4. Разработка драйверов устройств
- Драйвер клавиатуры
- Драйвер экрана (текстовый/графический режим)
- Драйвер диска
- Драйвер таймера

### 1.5. Реализация файловой системы
- Поддержка простой файловой системы (например, FAT12/16)
- Операции чтения/записи файлов
- Управление каталогами

### 1.6. Создание пользовательского интерфейса
- Текстовая консоль
- Базовые команды оболочки
- Простой текстовый редактор

### 2.1. Прием запросов

### 2.2. Генерация кода

- Использование шаблонов для стандартных компонентов
- Автоматическая адаптация кода под выбранную архитектуру

### 2.3. Представление результатов
- Генерация отчета о созданных компонентах
- Предоставление исходного кода с комментариями
- Создание документации по сборке и запуску
