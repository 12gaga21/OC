# Визуальный стиль "Славянский Терминал"

## Описание
Оригинальный визуальный стиль операционной системы "UNIVERSAL ASM CORE", вдохновленный эстетикой дореволюционной России и Российской Империи.

## Цветовая палитра
- **Основной**: Золотой (желтый) текст на темно-синем фоне (0x1E)
  - Символизирует императорское знамя
- **Особые сообщения**: Белый на синем (0x1F)
- **Предупреждения**: Золотой на черном (0x0E)

## Типографика
- Используется шрифт VGA по умолчанию с кодировкой CP866
- Поддержка русской дореформенной орфографии (съ, ѣ, і, ъ)
- Символы: † (крест) для особых заголовков

## Формат сообщений
```
† ВСЕМОСТИВЫЙ ГОСУДАРЬ †
[КОМПОНЕНТ] Сообщение в дореформенной орфографии
```

## Примеры использования
- Загрузчик: "Переходъ въ защищенный режимъ..."
- Оболочка: "[СЛАВЯНСКІЙ ТЕРМИНАЛЪ]> "
- Ошибки: "Невѣдомая команда"

## Юридическая информация
Данный стиль является оригинальной разработкой и не нарушает авторских прав:
- Не использует защищенные товарные знаки или символы
- Вдохновлен историческими интерфейсами и общедоступной эстетикой
- Все тексты являются оригинальными формулировками

## Технические детали
- Совместимость: Любой VGA-совместимый адаптер
- Кодировка: CP866 (DOS Russian) с расширением до дореформенных букв
- Разрешение: 80x25 текстовый режим

## История стиля
Стиль разработан как альтернатива "индустриальному терминалу" для придания системе уникального национального характера без нарушения авторских прав.
