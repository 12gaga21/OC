# Визуальный стиль "UNIVERSAL ASM CORE"

## Описание стиля
Операционная система использует оригинальный визуальный стиль **"Индустриальный Терминал"**, вдохновленный классическими компьютерными интерфейсами 1970-80х годов и эстетикой ретро-футуризма.

## Цветовая палитра
- **Основной цвет**: Зеленый текст на черном фоне (0x0A)
- **Предупреждения**: Белый текст на черном фоне (0x0F)
- **Фон элементов**: Темно-зеленый (0x02)

## Типографика
- Шрифт: VGA Text Mode 8x16 (встроенный)
- Разрешение: 80x25 символов
- Кодировка: CP866 (поддержка русского языка)

## Элементы интерфейса

### Загрузчик
```
[ЗАГРУЗЧИК] Инициализация системы...
[СИСТЕМА] Проверка аппаратных ресурсов...
[ЯДРО] Подготовка к загрузке...
```

### Ядро при запуске
```
[СИСТЕМА] Инициализация визуального протокола...
[ЯДРО] Синхронизация матрицы дисплея...
[ГОТОВО] Терминал активен.
```

### Командная строка
```
[ТЕРМИНАЛ]> 
```

## Принципы дизайна
1. **Минимализм** - только необходимая информация
2. **Консистентность** - единый формат сообщений `[КОМПОНЕНТ] Сообщение`
3. **Читаемость** - высокий контраст, моноширинный шрифт
4. **Универсальность** - работает на любом VGA-совместимом оборудовании

## Совместимость
- Не использует защищенные авторским правом элементы
- Оригинальный дизайн без отсылок к конкретным франшизам
- Полностью открытая реализация

## Технические детали
- Все текстовые сообщения хранятся в ASCII/CP866
- Цвета задаются через атрибуты VGA
- Поддержка прокрутки и курсора
