## 🌈 Цветовая палитра

| Код | Название | RGB | Применение |
|-----|----------|-----|------------|
| `0x0A` | **Изумрудный Люминофор** | Зеленый на черном | Основной текст терминала |
| `0x1E` | **Церковное Золото** | Желтый на синем | Рамки, заголовки, особые элементы |
| `0x04` | **Кровь Ошибки** | Красный на черном | Сообщения об ошибках |
| `0x0F` | **Белая Святость** | Белый на черном | Особо важные сообщения |
| `0x02` | **Тень** | Темно-зеленый на черном | Второстепенный текст |

### Примеры цветов в ASCII:

```

```

## 📐 Элементы интерфейса

### 1. Золотые рамки (Box Drawing)

Используются символы псевдографики CP866 для создания церковных орнаментов:

```
╔════════════════════════════╗
║ † ЗАГОЛОВОКЪ ВЪ РАМКѢ †   ║
╚════════════════════════════╝
```

**Символы:**
- `╔` (0xD5) - Верхний левый угол
- `╗` (0xB8) - Верхний правый угол
- `╚` (0xC9) - Нижний левый угол
- `╝` (0xBC) - Нижний правый угол
- `═` (0xCD) - Горизонтальная линия
- `║` (0xBA) - Вертикальная линия

### 2. Религиозные символы

- `†` - Крест (используется для заголовков и важных сообщений)
- `☼` (0x06) - Солнце/Символ божественного света

### 3. Индикаторы прогресса

```
Загрузка ядра... [██████████] 100%
Инициализация.. [▓▓▓▓▓▓░░░░] 60%
```

## 🔧 Техническая реализация

### Файлы модулей

| Файл | Описание |
|------|----------|
| `vga.asm` | Драйвер VGA с поддержкой цветовой палитры |
| `shell.asm` | Оболочка с стилизованным приглашением |
| `boot.asm` | Загрузчик с ритуальными сообщениями |
| `kernel.asm` | Ядро с интеграцией стилевых элементов |

### API функций

```asm
; Установка цвета
mov al, 0x0A          ; Зеленый на черном
call vga_set_color

; Рисование золотой рамки
mov ax, 10            ; X
mov bx, 5             ; Y
mov cx, 60            ; Ширина
mov dx, 10            ; Высота
call vga_draw_box

; Печать ошибки
mov si, error_message
call vga_print_error
```

## 📜 Дореформенная орфографія

### Основные правила

1. **Буква "ѣ" (ять)** - используется вместо современного "е" в определенных позициях
   - Примѣръ: "бѣлый", "свѣтъ", "вѣра"

2. **Буква "і" (и краткое)** - перед гласными и в некоторых словах
   - Примѣръ: "Россійскій", "имперія", "исторія"

3. **Буква "ъ" (ер)** - на конце слов после согласных
   - Примѣръ: "терминалъ", "компьютеръ", "каталогъ"

4. **Буква "ѳ" (фита)** - в словах греческого происхождения
   - Примѣръ: "Ѳедоръ", "каѳедра"

### Словарь стиля

| Современный | Дореформенный |
|-------------|---------------|
| терминал | терминaлъ |
| система | систeма |
| память | памaть |
| ошибка | oшибка |
| успех | успѣхъ |
| благословение | благословенiе |

## 🎯 Принципы дизайна

### 1. Единство стиля
Все элементы интерфейса должны соответствовать общей концепции "техно-ритуала".

### 2. Иерархия информации
- **Золотые рамки** - заголовки и важные блоки
- **Зеленый текст** - основное содержание
- **Красный текст** - только ошибки и критические сообщения

### 3. Культурная аутентичность
Использование дореформенной орфографии должно быть последовательным и грамотным.

### 4. Производительность
Визуальные эффекты не должны замедлять работу системы.
