# Поддержка русского языка в ОС

## Обзор

ОС поддерживает отображение русского языка через кодировку **CP866** (стандартная кодировка для текстового режима VGA в DOS/BIOS).

## Реализованные компоненты

### 1. Модуль русской раскладки клавиатуры (`rus_layout.asm`)

**Файл:** `os/src/kernel/rus_layout.asm` (201 строка)

**Возможности:**
- Преобразование сканкодов в символы CP866
- Поддержка раскладки ЙЦУКЕН
- Переключение между латиницей и кириллицей
- Учёт регистра (Shift + CapsLock)
- Полные таблицы символов для нижнего и верхнего регистра

**API функции:**
```nasm
rus_layout_init              ; Инициализация модуля
rus_scan_to_cp866            ; Преобразование сканкода в CP866 символ
                             ; Вход: bl = сканкод, dh = Shift状态
                             ; Выход: al = символ CP866
rus_toggle_caps              ; Переключение CapsLock
rus_toggle_layout            ; Переключение раскладки (RU/EN)
rus_get_layout_state         ; Получить текущую раскладку (0=EN, 1=RU)
rus_set_shift_state          ; Установить состояние Shift
rus_get_shift_state          ; Получить состояние Shift
```

## Интеграция с существующим кодом

### Обновление обработчика клавиатуры

Для использования русской раскладки необходимо обновить `keyboard.asm`:

```nasm
; В keyboard_handler добавить после получения сканкода:
extern rus_scan_to_cp866
extern rus_toggle_layout
extern rus_get_shift_state

; Проверка на специальную клавишу для переключения раскладки (например, правый Ctrl)
cmp bl, 0x1D  ; Правый Ctrl
je .toggle_layout

; Получение состояния Shift
call rus_get_shift_state
mov dh, al

; Преобразование с учётом русской раскладки
call rus_scan_to_cp866
cmp al, 0
je .skip_store  ; Если 0, символ не поддерживается

; Использование al вместо оригинального ASCII
```

### Переключение раскладки

Горячие клавиши для переключения:
- **Правый Ctrl** - переключение RU/EN
- **CapsLock** - переключение регистра
- **Shift** - временный верхний регистр

## Пример использования в shell

```nasm
; В shell.asm при выводе приглашения:
mov si, shell_prompt_ru
call kprint

shell_prompt_ru:
    db 128, 132, 142, 143, 133, 153, 133, 148, 153, 133, '>', 0
    ; "Приглашение>" в CP866
```

## Тестирование

1. Скомпилировать ОС с новым модулем
2. Запустить в QEMU
3. Нажать правый Ctrl для переключения на русскую раскладку
4. Вводить русский текст в shell
5. Проверить отображение всех букв

## Ограничения

- Только кодировка CP866 (не поддерживает UTF-8)
- Только текстовый режим VGA (80x25)
- Нет поддержки ё/Ё в базовой реализации
- Только раскладка ЙЦУКЕН (нет других вариантов)
