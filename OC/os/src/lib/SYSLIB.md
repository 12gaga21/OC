# СВЯЩЕННАЯ БИБЛІОТЕКА (SYSLIB)

## Описаніе
**Священная Библіотека** — это наборъ базовыхъ функцийъ, написанныхъ на чистомъ ассемблерѣ NASM, для использования въ программахъ операционной системы "Золотой Механикусъ".

### Особенности:
- **100% Ассемблеръ**: Никакихъ зависимостей отъ C или другихъ языковъ
- **Оптимизирована**: Минимальный размеръ и максимальная скорость
- **Стиль Злато-Изумрудный**: Встроенная поддержка фирменной цвѣтовой схемы
- **Дореформенная орфографія**: Комментарии и документация въ старорусскомъ стилѣ

## Функции Библіотеки

### Инициализація
| Функция | Описаніе |
|---------|----------|
| `sl_init()` | Инициализація библіотеки (очистка экрана, сбросъ курсора) |

### Выводъ на экранъ
| Функция | Описаніе | Параметры |
|---------|----------|-----------|
| `sl_print_string(esi)` | Выводъ NULL-терминированной строки | ESI = адресъ строки |
| `sl_print_hex(eax)` | Выводъ числа в HEX форматѣ | EAX = число |
| `sl_print_dec(eax)` | Выводъ числа в DEC форматѣ | EAX = число |
| `sl_clear_screen()` | Очистка экрана | - |
| `sl_set_cursor(ax, bx)` | Установка курсора | AX = X, BX = Y |
| `sl_get_cursor()` | Полученіе позиціи курсора | Возвратъ: AX=X, BX=Y |

### Работа съ памятью
| Функция | Описаніе | Параметры |
|---------|----------|-----------|
| `sl_memcpy(esi, edi, ecx)` | Копированіе памяти | ESI=источникъ, EDI=назначеніе, ECX=размѣръ |
| `sl_memset(edi, al, ecx)` | Заполненіе памяти | EDI=адресъ, AL=значеніе, ECX=количество |
| `sl_strlen(esi)` | Длина строки | ESI=строка, возвратъ EAX=длина |
| `sl_strcmp(esi, edi)` | Сравненіе строкъ | ESI=строка1, EDI=строка2, возвратъ 0=равны |
| `sl_strcpy(edi, esi)` | Копированіе строки | EDI=назначеніе, ESI=источникъ |

### Утилиты
| Функция | Описаніе | Параметры |
|---------|----------|-----------|
| `sl_delay(ecx)` | Задержка на такты | ECX = количество цикловъ |
| `sl_rand()` | Случайное число | Возвратъ EAX = 0..32767 |
| `sl_rand_seed(eax)` | Инициализація ГСЧ | EAX = сѣмя |

## Цвѣтовая схема (Злато-Изумрудный Ритуалъ)

```nasm
COLOR_TEXT:       equ 0x0A  ; Изумрудный (зеленый люминофоръ)
COLOR_BORDER:     equ 0x1E  ; Золото на синемъ (церковный стиль)
COLOR_ERROR:      equ 0x0C  ; Красный (ошибки Инквизиціи)
COLOR_SUCCESS:    equ 0x0A  ; Зеленый (успѣхъ)
COLOR_INFO:       equ 0x1F  ; Бѣлый на синемъ (информація)
```

## Примѣръ использованія

```nasm
; Примѣръ программы съ использованиемъ Священной Библіотеки
BITS 32
EXTERN sl_init, sl_print_string, sl_clear_screen, sl_print_hex
GLOBAL _start

SECTION .data
    msg db '† Слава Омниссіи! †', 0
    num dd 0xDEADBEEF

SECTION .text
_start:
    call sl_init              ; Инициализація
    
    mov esi, msg
    call sl_print_string      ; Выводъ приветствія
    
    mov eax, [num]
    call sl_print_hex         ; Выводъ числа в HEX
    
    ; Безконечный циклъ
.halt:
    jmp .halt
```

## Компиляція и линковка

### Сборка библіотеки:
```bash
nasm -f elf32 syslib.asm -o syslib.o
ar rcs libsyslib.a syslib.o
```

### Сборка программы съ библіотекой:
```bash
nasm -f elf32 myprog.asm -o myprog.o
ld -m elf_i386 -T myprog.ld myprog.o -L. -lsyslib -o myprog.bin
```
