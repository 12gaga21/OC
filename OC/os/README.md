# Операционная система на языке ассемблере

## Описание

Этот проект представляет собой операционную систему, полностью написанную на языке ассемблере. ОС включает в себя базовые драйверы устройств и может быть собрана и запущена в эмуляторе QEMU.

## Основные функции

- Загрузчик, написанный на ассемблере NASM
- Ядро операционной системы, полностью написанное на ассемблере
- Базовые драйверы устройств, полностью реализованные на ассемблере:
  - Драйвер клавиатуры (PS/2)
  - Драйвер таймера (PIT)
  - Драйвер последовательного порта (COM1)
  - Расширенный драйвер VGA
- Файловая система FAT16 с поддержкой чтения/записи файлов и директорий
- Интерактивная оболочка для работы с файловой системой
- Полнофункциональные сетевые возможности (TCP/IP стек, ping, HTTP клиент, DNS клиент)
- Многозадачность на уровне ядра
- Система управления памятью
- Системные вызовы

## Реализованные компоненты

### Загрузчик
- Инициализация 16-битного реального режима
- Вывод приветственного сообщения через BIOS
- Включение линии A20
- Переключение в 32-битный защищенный режим
- Настройка GDT (Global Descriptor Table)
- Создание стека в 32-битном режиме

### Ядро операционной системы
- Точка входа в 32-битном режиме
- Базовая система прерываний
- Обработчики исключений
- Основная функция ядра (kmain) на ассемблере
- Вспомогательные функции ядра (работа со строками, памятью)
- Функции работы с числами (преобразование в строки и обратно)
- Функции получения информации о системе

### Многозадачность
- Поддержка до 16 задач
- Переключение между задачами
- Создание новых задач
- Управление состоянием задач

### Система управления памятью
- Выделение и освобождение памяти
- Информация о состоянии памяти

### Системные вызовы
- Обработчики системных вызовов
- Поддержка вызовов печати, создания задач, завершения задач

### Драйвер клавиатуры (keyboard.asm)
- Обработка прерываний клавиатуры (IRQ1)
- Чтение сканкодов с порта 0x60
- Отправка сигнала EOI контроллеру прерываний
- Хранение последнего сканкода в памяти

### Драйвер таймера (timer.asm)
- Настройка частоты таймера (100 Гц)
- Обработка прерываний таймера (IRQ0)
- Подсчет тиков с момента запуска системы
- Отправка сигнала EOI контроллеру прерываний

### Драйвер последовательного порта (serial.asm)
- Инициализация последовательного порта COM1
- Настройка скорости 38400 бод
- 8 бит данных, 1 стоп-бит, без четности
- Поддержка FIFO буферизации
- Функции отправки и получения отдельных байтов
- Функция отправки строк

### Расширенный драйвер VGA (vga.asm)
- Инициализация VGA режима 80x25 текстовый цветной
- Вывод символов и строк в любую позицию экрана
- Поддержка цветного текста
- Прокрутка экрана
- Управление позицией курсора
- Обработка специальных символов (backspace, tab, newline)

### Файловая система FAT16
- Чтение параметров файловой системы из BPB (BIOS Parameter Block)
- Доступ к данным через BIOS INT 13h
- Поддержка формата имен файлов 8.3
- Чтение корневой директории
- Поиск файлов по имени
- Чтение содержимого файлов
- Работа с FAT таблицей для навигации по кластерам
- Преобразование номеров кластеров в номера секторов
- Работа с подкаталогами
- Навигация по иерархии директорий
- Запись данных на диск
- Создание новых файлов и директорий
- Удаление файлов и директорий

### Интерактивная оболочка (shell.asm)
- Вывод приглашения командной строки
- Чтение команд от пользователя
- Парсинг и выполнение команд
- Поддержка команд: ls, cd, cat, sysinfo

### Сетевые возможности
- Драйвер сетевой карты
- Базовый TCP/IP стек
- Поддержка протоколов Ethernet, IP, TCP, UDP, ICMP, ARP
- Создание сетевых заголовков
- Обработка входящих сетевых пакетов
- Отправка исходящих сетевых пакетов
- Базовые сетевые функции
- Поддержка ping
- Поддержка HTTP запросов
- Поддержка разрешения DNS имен

## Инструкция по сборке и запуску

1. Установите необходимые зависимости:
   - NASM (ассемблер)
   - QEMU (эмулятор)

2. Перейдите в каталог сборки:
   ```bash
   cd build
   ```

3. Запустите скрипт сборки:
   ```bash
   ./build.sh
   ```

4. Запустите ОС в эмуляторе QEMU:
   ```bash
   qemu-system-i386 -fda build/os.img
   ```

## Требования к окружению

- NASM 2.15 или выше
- QEMU 6.0 или выше
- Linux или macOS (для сборки)
